pebbleDictionaryData:

Serializes the receiver into a Pebble dict.

- (NSData *)pebbleDictionaryData:(NSError *__autoreleasing *)error

Parameters

error[out]

Pointer to an NSError that will be set after the method has returned in case there was an error. Possible error codes: PBErrorCodeDictionaryUnsupportedKeyClass, PBErrorCodeDictionaryUnsupportedValueClass or PBErrorCodeDictionaryUnsupportedValueClass.

Discussion

Serializes the receiver into a Pebble dict.

Note: The receiver dictionary must comply to the following constraints: * The keys must be instances of NSNumber. Their -unsignedLongValue return value will be used as final key. * The values must be instances of either NSString, NSData or NSNumber. Use the NSNumber (stdint) category to specify the signedness and width of NSNumbers.
